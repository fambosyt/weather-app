# 🌤️ Wetter App

Eine einfache Wetter-App, die dir das aktuelle Wetter für jede Stadt der Welt anzeigt – mit der OpenWeatherMap API.

## 🚀 Features

- Aktuelle Temperatur, Wetterbeschreibung, Icon
- Deutsche Sprache
- Responsive & clean UI
